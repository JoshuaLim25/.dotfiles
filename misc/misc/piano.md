# Sightreading
- Notice that the keyboard is grouped like so: 3 white keys with 2 black keys in between, then 4 white keys with 3 black in between. The leftmost key of the first group is C; and for the second group, F. Use those as reference points.
- For those with some experience but still have trouble navigating the board: practice your arpeggios. At this point, rather than aiming to know where each and every key is, it's more realistic and efficient to know how _far_ the intervals are. Things like the handspan of an octave, the width of thirds and fifths, etc. should be muscle memory. This is also the case with sight-reading.
- For those who want to improve their proprioception: practice leapy pieces. The opening bars of La Campanella for right hand, Rach's Op. 23 No. 5, stride jazz pieces, Chopin's Op. 25 No. 12 for hands separate then together, Chopin's Op. 25 No. 4, Liszt's Un Sospiro, Bach's Goldberg Variation 5, etc.

# FAQ
- [note names]()
We can talk for hours about this, but I think the answer you need right now is this: C# is the same as Db, D# is the same as Eb, and so on. Each black key has two equivalent names. You _could_ spell C minor as C-D#-G, and you would be playing the correct notes on the piano.

---

Now, there _is_ a reason why we use Eb instead of D#, and it is that we want scales to have consistent note names. The C major chord is basically built out of the C major scale

```
C D E F G A B
```

by taking the 1st, 3rd and 5th notes. Similarly, the C minor chord comes from the C minor scale

```
C D Eb F G Ab Bb
```

(I'm oversimplifying a bit), again by taking the 1st, 3rd and 5th notes. We _could_ write this scale as

```
C D D# F G G# A#
```

and it would contain the same notes, but it would be a bit weird: it has two D's, two G's, and no E or B. This is a problem not only because the first version looks prettier, but because it obscures the relationship between the two scales. By using flats, we can see that the minor scale takes the major scale and lowers the 3rd, 6th and 7th by a semitone. In some other scales, we would use sharps instead of flats, for the same reason. Don't get overwhelmed by this right now, though: I'm just showing that there's a reason for choosing sharps or flats in each situation, I don't expect you to understand or remember everything.
